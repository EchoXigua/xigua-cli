## 将包变成全局的步骤

1. 先创建可执行的脚本
2. 配置package.json中的bin
3. npm link 链接到本地环境



link 相当于将本地模块链接到npm目录下，这个npm目录可以直接访问，所有当前包就可以直接访问了

```js
#! /usr/bin/env node  
//表示当前环境为node  固定写法

console.log('xg-cli')
```
